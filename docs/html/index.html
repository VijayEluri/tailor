Getting the software

There are two implementations of Tailor, one in python and one in java. The python version is better for command line use, while the java version has the advantage of a gui.

  • Tailor (python version)
  • Tailor (java version)

What does it do?

Tailor is a general tool for searching structures and measuring geometric features on the results. It will (eventually) be flexible enough to find a wide range of structural patterns - from backbone motifs to multi-chain binding sites.

The standard measurements that can be made are distances and angles, but custom measures can be defined by the user. Simple examples of how to do this are given here.

The java version will run anywhere straight from the jar file. However using the program may be easier if it is put in a new directory and run from there.
